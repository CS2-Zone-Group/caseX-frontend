# CaseX Frontend

Next.js frontend for CaseX CS2 Skins Marketplace

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- Axios

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```bash
cp .env.example .env.local
```

3. Configure environment variables

4. Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Steam OAuth authentication
- Marketplace with skin listings
- User profile and inventory
- O'zbek tili support
- Responsive design

## Development

```bash
# development
npm run dev

# build
npm run build

# production
npm run start
```
